# F1-Driver-Alpha-Framework
Performance intelligence platform analyzing 75 years of Formula 1 history, evaluating 850+ drivers across 70,000+ race and qualifying records using SQL, Power BI, and advanced analytics to identify the drivers of long-term competitive success.

**Business Problem**
Formula 1 performance is often judged using championships and race wins alone. This project develops a data-driven framework to evaluate driver performance across eras by combining teammate comparisons, qualifying effectiveness, championship efficiency, consistency, and career longevity into a unified intelligence platform.

**Dataset**
75 years of Formula 1 history
850+ drivers
70,000+ race and qualifying observations
Championships, podiums, wins, points, qualifying performance, teammate comparisons, constructors, circuits, and seasons

**Tools Used**
Jupyter
SQL
PowerBI
Excel

**Key Analytical Frameworks**
Driver Alpha Score
Championship Efficiency Analysis
Teammate Dominance Framework
Qualifying Performance Index
Career Consistency Assessment
Era-Normalized Performance Evaluation
